# ✨ Premium Web Utilities & Mini-Game Suite

> 반응형 디자인, 다크 모드, 그리고 마이크로 인터랙션이 포함된 10종의 웹 유틸리티 도구 및 미니 게임 모음집입니다.

<br />

## 🌟 프로젝트 소개
본 프로젝트는 현대적이고 세련된 UI/UX 디자인을 기반으로 일상에서 유용하게 사용할 수 있는 **10가지의 웹 유틸리티 도구**와 **몰입감 넘치는 미니 게임**을 제공하는 웹 사이트입니다. 
글래스모피즘(Glassmorphism) 스타일과 부드러운 그라데이션, 그리고 반응형 레이아웃을 적극 반영하여 제작되었습니다.

---

## 🛠️ 기술 스택 (Tech Stack)
* **Core**: React (Vite)
* **Styling**: Tailwind CSS & Vanilla CSS (Custom Keyframe Animations)
* **Icons**: React Icons
* **Deployment**: GitHub Pages / Vercel / Netlify (배포 예정)

## 📱 주요 기능 & 유틸리티 구성
### 1. 10가지 웹 유틸리티 (`UtilitiesSuite`)
* **디지털시계 & 타이머**: 스톱워치, 알림 타이머 기능이 포함된 정밀한 시계
* **단위 변환기**: 길이, 무게, 온도 등 다채로운 단위 자동 계산기
* **금융 계산기**: 적금/예금 이자 계산 및 대출 상환 계획기
* **칼로리 & BMI 계산기**: 건강 관리를 위한 신체 지수 측정 도구
* **비밀번호 생성기**: 보안 수준 설정 및 무작위 고강도 비밀번호 생성
* **텍스트 카운터**: 글자 수, 단어 수, 공백 유무 분석 및 대소문자 변환
* **할 일 목록 (To-Do List)**: 카테고리 분류 및 완료 상태 관리가 가능한 세련된 플래너
* **색상 코드 추출기**: RGB, HEX, HSL 상호 변환 및 팔레트 추출 도구
* **할인율 계산기**: 원가, 할인율 설정에 따른 최종가 및 절약 금액 계산기
* **랜덤 추천기 (Decision Maker)**: 메뉴나 결정을 내리기 어려울 때 항목을 입력하여 무작위로 추첨해 주는 도구

### 2. 엔터테인먼트 (`MiniGame`)
* **미니 게임**: 재미와 몰입을 더해주는 웹 브라우저용 반응형 미니 게임
